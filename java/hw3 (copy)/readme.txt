Требуется реализовать новый тип коллекции - мультимножество.

В SVN выложен файл hw3/multiset.jar, содержащий классы и исходники, необходимые для выполнения задания.

Требования:

Ваш класс должен реализовывать интерфейс ru.compscicenter.java2013.collections.MultiSet и удовлетворять его спецификации (см. javadoc к этому интерфейсу). Операции, помеченные в Javadoc как опциональные, тоже должны быть реализованы.

Все файлы, относящиеся к этому заданию, должны находиться в директории hw3 или поддиректориях и должны быть закомичены в SVN. Через сайт присылать ничего не надо. В SVN не должны комититься результаты сборки и файлы среды разработки. Нужны только исходники решения и скрипт сборки.

Вместе с исходниками решения должен быть скрипт (ant, maven или gradle - на выбор) для сборки классов и JAR-архива. Скрипт должен работать в любом окружении. Результатом сборки должен быть архив со скомпилированными классами под названием multiset-impl.jar, расположенный непосредственно в директории hw3.

Включать в архив multiset-impl.jar классы из multiset.jar не требуется.

Оцениваться будет: корректность работы мультимножества с различным содержимым, эффективность реализации, качество кода, скрипт сборки.
